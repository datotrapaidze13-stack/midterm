# თარჯიმანი (Translator)

მარტივი კონსოლის თარჯიმანი, რომელიც თარგმანებს ინახავს ტექსტურ ფაილებში
(ლექსიკონებში).

## აღწერა

მომხმარებელი ირჩევს ენების წყვილს (ქართული-ინგლისური, ქართული-რუსული,
ინგლისური-ქართული, რუსული-ქართული), შემდეგ შეყავს სიტყვა ან მოკლე ფრაზა.
პროგრამა ეძებს თარგმანს შესაბამის ლექსიკონის ფაილში (`dictionaries/` საქაღალდე)
და აჩვენებს მას. თუ სიტყვა ლექსიკონში არ მოიძებნა, პროგრამა სთავაზობს
მომხმარებელს დაამატოს ახალი თარგმანი, რომელიც შემდეგ ინახება ტექსტურ ფაილში
შემდგომი გამოსაყენებლად.

## ლექსიკონის ფორმატი

თითო ხაზზე: `სიტყვა=თარგმანი`

## გაშვება

```
python translator.py
```

მოითხოვს Python 3-ს.

# თარჯიმანი (Translator)

მარტივი კონსოლის თარჯიმანი, რომელიც თარგმანებს ინახავს JSON ფაილებში
(ლექსიკონებში).

## აღწერა

მომხმარებელი ირჩევს ენების წყვილს (ქართული-ინგლისური, ქართული-რუსული,
ინგლისური-ქართული, რუსული-ქართული), შემდეგ შეჰყავს სიტყვა ან მოკლე ფრაზა.
პროგრამა ეძებს თარგმანს შესაბამის ლექსიკონის JSON ფაილში (`dictionaries/`
საქაღალდე) და აჩვენებს მას. თუ სიტყვა ლექსიკონში არ მოიძებნა, პროგრამა
სთავაზობს მომხმარებელს დაამატოს ახალი თარგმანი, რომელიც შემდეგ ინახება
JSON ფაილში შემდგომ გამოსაყენებლად.

## ლექსიკონის ფორმატი

თითოეული ლექსიკონი არის JSON ობიექტი, key-value წყვილებით:

```json
{
    "სიტყვა": "თარგმანი"
}
```

## გაშვება

```
python translator.py
```

მოითხოვს Python 3-ს.

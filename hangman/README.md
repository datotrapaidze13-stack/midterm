# Hangman

სიტყვების გამოცნობის თამაში კონსოლში.

## აღწერა

პროგრამა შემთხვევით ირჩევს სიტყვას წინასწარ განსაზღვრული 5 სიტყვისგან შემდგარი
ლექსიკონიდან (`WORDS`, სადაც თითოეულ სიტყვას თან ახლავს მოკლე მინიშნება/ჰინტი) და
აჩვენებს მას ქვედა ტირეების სახით (თითო ასოზე ერთი ტირე), მინიშნებასთან ერთად.
მომხმარებელი ცდილობს გამოიცნოს სიტყვის ასოები. სწორად გამოცნობილი ასოები იხსნება
სიტყვაში, არასწორი მცდელობები ამცირებს დარჩენილ მცდელობათა რაოდენობას (სულ 6).
თამაში მთავრდება, როცა მომხმარებელი გამოიცნობს მთელ სიტყვას ან ამოეწურება
მცდელობები.

## გაშვება

```
python hangman.py
```

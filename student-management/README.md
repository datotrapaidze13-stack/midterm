# სტუდენტების მართვის სისტემა

კონსოლის აპლიკაცია სტუდენტების მართვისთვის, ობიექტზე ორიენტირებული
პროგრამირების პრინციპებით.

## აღწერა

პროგრამა საშუალებას აძლევს მომხმარებელს:
- დაამატოს ახალი სტუდენტი (სახელი, გვარი, სიის ნომერი, შეფასება)
- ნახოს ყველა დამატებული სტუდენტი
- მოძებნოს სტუდენტი სიის ნომრით
- განაახლოს არსებული სტუდენტის შეფასება
- ნებისმიერ დროს, ნებისმიერი კითხვისას გამოვიდეს პროგრამიდან `/q` ბრძანებით

## OOP პრინციპები

- **ინკაფსულაცია** — `Student`-ის ატრიბუტები (`first_name`, `last_name`,
  `roll_number`, `grade`) დაცულია `@property`/setter-ებით, რომლებიც
  ავალიდირებენ მონაცემს ყოველი ცვლილებისას (მაგ. სიის ნომერი უნდა იყოს
  დადებითი მთელი რიცხვი, შეფასება — ერთ-ერთი A/B/C/D/F, სახელი/გვარი —
  მხოლოდ ასოებისგან შემდგარი).
- **მემკვიდრეობა** — `Student` იღებს მემკვიდრეობით `Person` კლასს, საიდანაც
  სახელი/გვარის ლოგიკას იყენებს.
- **პოლიმორფიზმი** — `Student` თავისებურად გადაწერს (override) `Person`-ის
  `__str__` მეთოდს, საკუთარი ფორმატით.

## ვალიდაცია

- სახელი და გვარი არ შეიძლება იყოს ცარიელი და უნდა შეიცავდეს მხოლოდ ასოებს
  (ციფრების ან სიმბოლოების გარეშე); თუ ორივე ერთ ველში (space-ით გამოყოფილი)
  შეიყვანება, პროგრამა ცალკე გაფრთხილებას აჩვენებს
- სიის ნომერი უნდა იყოს დადებითი მთელი რიცხვი და უნიკალური
- შეფასება უნდა იყოს ერთ-ერთი: A, B, C, D, F

## მუდმივი შენახვა

ყველა სტუდენტი ავტომატურად ინახება `students.json` ფაილში, იმავე საქაღალდეში,
სადაც სკრიპტია (მდებარეობა დამოუკიდებელია იმისგან, საიდანაც სკრიპტი გაეშვება).
პროგრამის ხელახლა გაშვებისას ეს მონაცემები ავტომატურად იტვირთება — ინფორმაცია
არ იკარგება პროგრამის დახურვის შემდეგ.

## გაშვება

```
python student_management.py
```

მოითხოვს Python 3-ს. ნებისმიერ კითხვაზე `/q`-ის შეყვანით პროგრამა მაშინვე
დასრულდება.
